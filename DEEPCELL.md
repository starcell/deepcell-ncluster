# Deepcell based on Nauta

Deepcell은 Intel Nauta를 기반으로 Deep Learning 전용 클러스터 시스템 구축에 필요한 기능이 추가되었다.
 
nauta관련 정보는 이 문서 참고 : https://intelai.github.io/nauta/


## 추가 기능 : 
Deepcell EDU Server 구축을 위한 edu template 추가
